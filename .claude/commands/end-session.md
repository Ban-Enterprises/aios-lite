End the session and save state to context files.

## Protocol

### 1. Update SESSION-STATE.md
Read `.context/SESSION-STATE.md` and update:
- Move current state to "Previous session"
- Fill in:
  - **What was done** — list of specific things
  - **Incomplete** — what needs to continue
  - **Next session — start with** — 2-3 specific actions

### 2. Update CHANGELOG.md
Add entry at the top with today's date:
- What changed
- Why

### 3. Update DECISIONS.md (if applicable)
- New decisions -> add to "Made"
- New questions -> add to "Open questions"

### 4. Update ROOT.md (if applicable)
- Did the priority change? Update it.

### 5. Confirm
Print: "Session closed. State saved. See you next time!"
