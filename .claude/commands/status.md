Show a quick status overview. Read minimum files, give maximum info.

## Protocol

### 1. Read
- `.context/ROOT.md` — priorities
- `.context/SESSION-STATE.md` — what happened recently
- `.context/DECISIONS.md` — open questions only

### 2. Print

```
STATUS — [date]

PRIORITY:
[from ROOT.md]

LAST SESSION:
[from SESSION-STATE.md — what was done + what's incomplete]

OPEN QUESTIONS:
[from DECISIONS.md — pending list]

NEXT STEPS:
[from SESSION-STATE.md — next actions]
```

### 3. Don't load anything else
Quick report. Max 60 lines of context.
