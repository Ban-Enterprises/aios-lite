# Research

> Find information on any topic. Search the web, analyze, summarize.

## Variables

topic: $ARGUMENTS

---

## Instructions

1. Read .context/ROOT.md — understand the user's context
2. Search the web (WebSearch, WebFetch) for information on: $ARGUMENTS
3. Collect key sources and facts
4. Summarize in an accessible way

---

## Output

Save to `outputs/research-YYYY-MM-DD-{topic}.md`:

```markdown
# Research: {Topic}

**Date:** YYYY-MM-DD
**Question:** {what we researched}

## Key findings
{5-7 points — the most important facts}

## Details
{Expansion of each finding with context}

## Sources
{List of links and sources}

## What this means for you
{How this connects to the user's situation}
```
